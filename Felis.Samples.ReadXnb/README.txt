このサンプル アプリケーションでは、XNA コンテンツ プロジェクトである
Felis.Sample.ReadXnbContent プロジェクトを参照します。
Felis.Sample.ReadXnbContent プロジェクトは、ビルド時に Content ディレクトリへ
.xnb ファイルを出力するようにビルド定義されています。

なお、このサンプル アプリケーションからは "dude" アセットをロードしますが、
"dude" アセットに対応する .fbx および .tga ファイルは GitHub のリポジトリに
含めていません。Felis.Sample.ReadXnbContent プロジェクトの README.txt を参考に
XNA Shadow Mapping サンプルより手動で配備して下さい。
